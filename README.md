## Usage

### Image pull

```bash
   podman pull quay.io/laraujo/tools:ocp-tools-latest
```

#### Run
```bash
   podman run -it --rm \
       --name ocp-tools \
       --net=host \
       --pid=host \
       --ipc=host \
       --cap-add=SYS_PTRACE \
       quay.io/laraujo/tools:ocp-tools-latest
```
